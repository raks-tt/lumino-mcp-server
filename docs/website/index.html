Get Started

Choose your preferred installation method and get up and running in minutes.

Simply paste this prompt into Claude Code and it will set everything up for you:

Provisioning Prompt
Provision the Lumino MCP server as a project-local MCP integration: 1. Clone the repository: git clone https://github.com/spre-sre/lumino-mcp-server.git 2. Install Python dependencies using uv: cd lumino-mcp-server && uv sync 3. Create .mcp.json in the current project root with this configuration. Replace <ABSOLUTE_PATH_TO_LUMINO> with the actual absolute path: { "mcpServers": { "lumino": { "type": "stdio", "command": "<ABSOLUTE_PATH_TO_LUMINO>/.venv/bin/python", "args": ["<ABSOLUTE_PATH_TO_LUMINO>/main.py"], "env": { "PYTHONUNBUFFERED": "1" } } } } 4. After creating .mcp.json, restart Claude Code in this project directory.

Install using MCPM, the MCP server package manager:

bash
# Install LUMINO MCP Server mcpm install @spre-sre/lumino-mcp-server # For a specific client mcpm install @spre-sre/lumino-mcp-server --claude mcpm install @spre-sre/lumino-mcp-server --gemini # Verify installation mcpm list

Clone the repository and use uv for dependency management:

bash
# Clone the repository git clone https://github.com/spre-sre/lumino-mcp-server.git cd lumino-mcp-server # Install dependencies uv sync # Run the server uv run python main.py

Traditional pip installation with virtual environment:

bash
# Clone the repository git clone https://github.com/spre-sre/lumino-mcp-server.git cd lumino-mcp-server # Create virtual environment python -m venv venv source venv/bin/activate # Install dependencies pip install -e . # Run the server python main.py

Run LUMINO in a container with your kubeconfig mounted:

bash
# Pull the container image podman pull quay.io/spre-sre/lumino-mcp-server:latest # Run with kubeconfig mount podman run -it --rm \ -v ~/.kube/config:/home/lumino/.kube/config:ro \ quay.io/spre-sre/lumino-mcp-server:latest

Integration Guides

Configure LUMINO with your preferred AI client.

Config file location:

  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows: %APPDATA%\Claude\claude_desktop_config.json
  • Linux: ~/.config/Claude/claude_desktop_config.json
claude_desktop_config.json
{ "mcpServers": { "lumino": { "command": "uv", "args": [ "run", "--directory", "/path/to/lumino-mcp-server", "python", "main.py" ], "env": { "PYTHONUNBUFFERED": "1" } } } }

Create .mcp.json in your project root:

.mcp.json
{ "mcpServers": { "lumino": { "type": "stdio", "command": "/path/to/lumino-mcp-server/.venv/bin/python", "args": ["/path/to/lumino-mcp-server/main.py"], "env": { "PYTHONUNBUFFERED": "1" } } } }

Verify: claude mcp list

Config file location:

  • macOS/Linux: ~/.config/gemini/mcp_servers.json
  • Windows: %APPDATA%\gemini\mcp_servers.json
mcp_servers.json
{ "mcpServers": { "lumino": { "command": "uv", "args": [ "run", "--directory", "/path/to/lumino-mcp-server", "python", "main.py" ], "env": { "PYTHONUNBUFFERED": "1" } } } }

Verify: gemini mcp list

Open Cursor Settings (Cmd+,) and search for "MCP", then add:

Cursor MCP Settings
{ "mcpServers": { "lumino": { "command": "uv", "args": [ "run", "--directory", "/path/to/lumino-mcp-server", "python", "main.py" ], "env": { "PYTHONUNBUFFERED": "1" } } } }

Restart Cursor IDE after adding the configuration.
